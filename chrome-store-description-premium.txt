Crunchy-Skip automatically clicks skip buttons on Crunchyroll the moment they appear, providing a seamless viewing experience without manual intervention.

KEYBOARD SHORTCUTS
• Press Alt+Shift+S (Option+Shift+S on Mac) to instantly toggle auto-skip on/off
• Press Alt+Shift+P (Option+Shift+P on Mac) to toggle Picture-in-Picture mode
• Dedicated Picture-in-Picture button beside the Settings icon (can be shown/hidden from popup)

AUTOMATIC DETECTION
• Detects Skip Intro, Skip Credits, Skip Recap, and Skip Preview buttons
• Works with all button styles and layouts used by Crunchyroll
• Handles embedded players and multiple page frames
• Smart debouncing prevents duplicate clicks
• Adjustable delay (0-2000ms) adapts to your connection speed

CUSTOMIZATION OPTIONS
• Enable or disable auto-skip feature
• Show or hide the Picture-in-Picture button
• Toggle Skip Intro and Skip Credits separately
• Adjust click delay to match your connection speed
• Enable debug logging for troubleshooting
• Settings automatically sync across all open Crunchyroll tabs
• Compatible with all keyboard layouts and languages

PRIVACY & SECURITY
• 100% open source - all code available on GitHub
• No tracking, analytics, or data collection
• No network requests except to Crunchyroll
• Minimal required permissions
• Your viewing habits remain completely private

KEY FEATURES
• Works immediately after installation with sensible defaults
• Lightweight and optimized for performance
• Reliable detection system handles Crunchyroll layout changes
• Regular updates maintain compatibility with site changes
• Active development and quick issue resolution

HOW TO USE
1. Install the extension from Chrome Web Store
2. Navigate to any Crunchyroll episode or movie
3. Skip buttons will be automatically clicked when they appear
4. Press Alt+Shift+S to toggle auto-skip on/off as needed
5. Access additional settings through the toolbar popup
